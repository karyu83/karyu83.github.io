MS Azure 기능 분석

2. Azure에서 지원하는 서비스

  • Azure IoT(Internet of Things)

    1. IoT Hub
      • 수십억 개의 IoT 자산 연결, 모니터링 및 관리
      • IoT Hub는 오픈 소스 SDK와 여러 프로토콜을 지원하는 유연한 개방형 CaaS(Cloud Platform as a Service)
    2. IoT Central
      • 클라우드 전문 지식이 필요 없는, SaaS의 단순한 IoT(사물 인터넷) 지원 경험
      • 대규모 IoT 자산의 연결, 모니터링 및 관리를 도와주는, 완전히 관리되는 글로벌 IoT SaaS(Software-as-a-Service) 솔루션
    3. Azure Sphere
      • 인터넷에 연결된 매우 안전한 마이크로 컨트롤러(MCU) 장치를 만들기 위한 새로운 솔루션
      • 고객이 선호하는 지능형 제품 및 환경을 만들어 대규모로 IoT(사물 인터넷)를 구현할 수 있는 가격으로 빠르게 출시할 수 있게 해주는 보안 및 연결 기반을 제공
    4. Azure Maps
      • 향상된 생산성을 위해 데이터에 위치 컨텍스트 제공
      • 동적 정보로 앱과 장치를 새로 고치기
      • 지리적 좌표 이상을 추적
    5. Event Grid
      • 대규모로 안정적인 이벤트 전송 이용
      • 완전히 관리되는 이벤트 라우팅 서비스
    6. Azure IoT Edge
      • 클라우드 인텔리전스 및 분석을 에지 장치로 확장
      • AI(인공 지능), Azure 서비스 및 사용자 지정 논리를 플랫폼 간 IoT 장치에 직접 배포하고 실행하여 로컬에 클라우드 인텔리전스를 제공하는 완전히 관리되는 서비스
    7. Azure IoT 솔루션 가속기
      • 템플릿을 사용하여 일반적인 IoT(사물 인터넷) 시나리오에 대한 완전히 사용자 지정 가능한 솔루션 작성
    8. Time Series Insights
      • IoT 장치에서 시계열 데이터 탐색 및 분석
    9. Azure Stream Analytics
      • 지능형 작업을 지원하는 주문형 실시간 분석 서비스
      • 간단한 SQL 형식 언어를 사용하여 여러 IoT 또는 비-IoT 데이터 스트림에 대해 병렬 실시간 분석
      • 모든 IoT 장치 및 게이트웨이의 데이터를 즉시 분석
      • 수초 내에 시작, 즉시 확장, 각 작업에 사용한 만큼만 비용 지불

  • Storage

    1. 디스크 Storage
      • 가상 머신을 위한 확장 가능하고 안전한 저장소 확보
      • 대기 시간은 짧고 처리량은 많아야 하는, I/O를 많이 사용하는 응용 프로그램에는 프리미엄 디스크 저장소(SSD)를 선택
      • 테스트 작업에는 표준 디스크 저장소(HDD)를 사용
      • 빠르게 크기를 조정할 수 있기 때문에 비용을 절감
    2. Blob 저장소
      • 비구조적 데이터를 위한 대규모 확장 가능한 보안 저장소
      • 엑사바이트 용량과 대규모 확장성으로 데이터 액세스가 필요한 빈도에 따라 핫, 쿨 또는 보관 계층에서 수백에서 수십억 개의 개체를 저장
      • 이미지, 비디오, 오디오, 문서 등 모든 형식의 구조화되지 않은 데이터를 쉽고 비용 효율적으로 저장
    3. 보관 저장소
      • 드물게 액세스하는 데이터 저장을 위한 업계 최고의 가격대
      • 미사용 데이터가 자동으로 암호화됨
      • 핫 및 쿨 저장소 계층과의 원활한 통합
    4. File Storage
      • 안전한 클라우드 파일 공유 확보
      • 온-프레미스 성능 및 기능을 위해 동기화를 사용하여 Azure로 서버를 확장
      • SMB 3.0 및 HTTPS를 사용하여 미사용 데이터 및 전송 중인 데이터 보호
      • 친숙한 도구를 사용하여 클라우드 파일 공유 관리 간소화
    5. Queue Storage
      • 앱 간 메시지 기반 통신을 위한 보안 저장소 확보
      • 대량 볼륨 클라우드 서비스를 위한 지속형 큐
      • 대규모 작업을 위한 단순하고 비용 효율적이며 내구성이 뛰어난 메시지 큐
      • .NET, Java, Android, C++, Node.js, PHP, Ruby 및 Python을 위한 다양한 클라이언트 라이브러리 지원
      • REST API를 통해 데이터에 액세스 가능
    6. Data Box
      • Azure로의 데이터 이동을 간소화하도록 설계된 어플라이언스
      • 테라바이트에서 페타바이트 단위로 데이터를 빠르고 쉽게 이동
      • 주문, 채우기, 반환의 모든 과정을 추적하는 Azure Portal
      • 많은 양의 데이터를 Azure로 전송하고 싶지만 시간이나 네트워크 가용성 또는 비용이 제한될 경우 Azure Data Box 서비스를 사용

  • 분석

    1. HDInsight
      • 엔터프라이즈용의 완전 관리형 전체 스펙트럼 오픈 소스 분석 서비스
      • Hadoop, Spark, Hive, LLAP, Kafka, Storm, R 등의 인기 있는 오픈 소스 프레임워크를 사용 가능
      • Azure HDInsight를 사용하면 ETL, Data Warehousing, Machine Learning, IoT 등의 다양한 시나리오를 이용 가능
      • 프로덕션 워크로드에 대한 종단 간 SLA를 제공
      • Big Data를 처리하기 위한 기본 인프라 솔루션
    2. Data Factory
      • 편리해진 대규모 하이브리드 데이터 통합
      • 데이터 파이프라인 만들기, 예약 및 모니터링
      • 여러 네이티브 데이터 커넥터와의 데이터 통합 가속화
      • 클라우드에서 완벽하게 관리되는 ETL 서비스
      • 빅 데이터 통합으로 데이터 웨어하우스 현대화
    3. Data Lake Analytics
      • 인텔리전스 작업에 힘을 실어주는, 주문형 분석 작업 서비스
      • 페타바이트급 데이터를 대상으로 대규모 병렬 데이터 변환과 처리 프로그램을 U-SQL, R, Python, .Net에서 손쉽게 개발하고 실행
      • 관리할 인프라를 별도로 둘 필요 없이 데이터를 언제든지 처리하고 즉시 확장하여 사용
      • 빅 데이터 프로그램을 간편하게 디버그하고 최적화
    4. Azure Analysis Services
      • 인프라 관리 없이 빠르게 시작
      • 비즈니스 요구 사항에 맞게 리소스 규모 조정
      • 즐겨 사용하는 데이터 시각화 도구를 사용하여 데이터 시각화
      • 안심하고 BI 솔루션 제어, 배포, 테스트 및 제공
    5. Log Analytics
      • 로그 데이터를 정보 및 동작으로 변환
      • 여러 소스의 로그 데이터를 빠르게 연결 및 수집
      • 강력한 기계 학습 구문을 사용하여 상관 관계 지정 및 분석
      • 표현력이 있는 언어를 사용하여 대화형으로 검색 및 쿼리
      • 특화된 관리 솔루션을 사용하여 깊은 통찰력 개발

    7. HDInsight용 Apache Storm
    8. HDInsight용 R 서버
