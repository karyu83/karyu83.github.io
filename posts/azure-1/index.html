MS Azure 소개 및 기능 분석

1. Azure 란

  • Azure 는 MS에서 서비스 하는 Cloud Service로 Amazon의 AWS와 치열하게 경쟁중이다. MS에서 주장하는 AWS보다 나은 점은 더 많은 지역을 커버하고 일관된 하이브리드 클라우드라는 점과 개발자 생산성이 좋다는 점이다.

2. Azure에서 지원하는 서비스

  • 컴퓨팅 : 기본적인 서비스로 가상화된 서버를 지원해주는 기능

    1. Virtual Machines
      • AWS의 EC2와 같은 가장 기본적인 가상 머신.
      • 빠른 시간에 구축할 수 있다.
    2. virtual Machine Scale Set
      • 수천 개의 VM을 몇 분 내에 만들도록 자동 크기 조정을 통해 고가용성 달성
    3. AKS(Azure Kubernetes Service)
      • 완전히 관리되는 서비스를 통해 Kubernetes의 배포, 관리 및 작업 간소화
      • Kubernetes(K8s) 란 디플로이 자동화, 스케일링, 컨테이너화된 애플리케이션의 관리를 위한 오픈 소스 시스템이다.
      • 목적은 여러 클러스터의 호스트 간에 애플리케이션 컨테이너의 배치, 스케일링, 운영을 자동화하기 위한 플랫폼을 제공하기 위함이다. 도커를 포함하여 일련의 컨테이너 도구들과 함께 동작한다.
    4. Functions
      • 이벤트 기반 서버리스 아키텍처를 사용하여 앱 개발 가속화
      • 서버없이 이벤트에 대해서 스크립트나 코드 조각을 실행하여 계산을 하는 서비스
    5. Service Fabric
      • Windows 또는 Linux에서 마이크로 서비스 개발 및 컨테이너 오케스트레이션
      • 마이크로 서비스 개발 및 애플리케이션 수명 주기 관리 간소화
    6. App Service
      • 완전히 관리되는 플랫폼을 사용하여 웹 및 모바일용 클라우드 앱 빠르게 만들기
      • 인프라를 관리할 필요 없이 선택한 프로그래밍 언어로 웹 응용 프로그램을 빌드하고 호스팅할 수 있다.
    7. Container Instances
      • 앱 컨테이너화 및 단일 명령으로 손쉽게 컨테이너 실행
      • 트래픽이 급증할 때 AKS 등과 연동하여 탄력적으로 확장 가능
    8. Batch
      • 수십, 수백 또는 수천 개의 가상 머신으로 확장할 수 있는 클라우드 규모의 작업 일정 및 계산 관리
    9. Cloud Services
      • 하드웨어 대신 앱에 집중하도록 도와주는 가용성과 확장성이 뛰어난 클라우드 응용 프로그램 및 API 만들기
      • 몇 분 안에 수천 개의 인스턴스 배포가 가능하며 Java, Node.js, PHP, Python, .NET 및 Ruby를 지원한다.
      • 수요에 맞춰 자동 크기 조정 및 비용 절감이 가능하다.

  • 데이터베이스

    1. Azure Cosmos DB
      • NoSQL 선택 사항에 대한 지원을 비롯한 업계 최고 성능 및 SLA로 전역 배포되는 다중 모델 데이터베이스.
      • IoT, 상거래, 게임 등 크기를 탄력적으로 조정해야하는 곳에 적합.
    2. Azure SQL Database
      • 신속하게 프로비전하고 즉석에서 확장 가능하며 기본 제공 인텔리전스 및 보안을 포함하는 완전히 관리되는 관계형 데이터베이스
      • SQL Server(MS-SQL)과 광범위한 호환성을 가져 바로 마이그레이션이 가능하다.
    3. Azure Database for MySQL
      • 추가 비용 없이 높은 가용성 및 보안이 기본 제공되는 확장성 있는 완전 관리형 MySQL 관계형 데이터베이스
    4. Azure Database for PostgreSQL
      • 추가 비용 없이 높은 가용성 및 보안이 기본 제공되는 확장성 있는 완전히 관리되는 PostreSQL 관계형 데이터베이스
    5. Azure Database for MariaDB
      • 추가 비용 없이 높은 가용성 및 보안이 기본 제공되는 확장성 있는 완전히 관리되는 MariaDB 관계형 데이터베이스
    6. SQL Data Warehouse
      • 추가 비용 없이 모든 수준에서 보안을 제공하는 탄력성이 뛰어난 완전 관리형 데이터 웨어하우스
      • 엔터프라이즈를 위한 빠르고 유연한 보안 분석 플랫폼
    7. Azure Database Migration Service
      • 응용 프로그램 코드를 변경하지 않고 클라우드로 데이터베이스 마이그레이션 지원
      • 온-프레미스 SQL Server 데이터베이스를 클라우드로 이전하기 위한 원활한 종단 간 솔루션
    8. Redis Cache
      • 높은 처리량과 일관되고 대기 시간이 짧은 데이터 액세스로 빠르고 확장 가능한 응용 프로그램을 실행가능
      • Redis Cache를 기반으로 하며, Microsoft에서 관리하여 Azure에 있는 모든 응용 프로그램에서 액세스할 수 있는 안전한 전용 Redis Cache에 대한 액세스를 제공
    9. Table Storage
      • 대량의 반구조화된 데이터 집합을 사용한 신속한 개발을 위한 NoSQL 키-값 저장소
      • 대규모 확장 가능 앱 구현, 유연한 데이터 스키마가 필요한 앱
      • JSON을 사용하여 데이터 직렬화, OData기반 쿼리 수행 가능
