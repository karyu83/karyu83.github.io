Azure 기능 분석 2편 - AI + Machine Learning

May 17, 2018 12:23 · 2225 words · 5 minute read cloud azure

MS Azure 기능 분석

2. Azure에서 지원하는 서비스

  • AI + Machine Learning

    1. Azure Databricks
      • 빠르고 쉬운 공동 작업 Apache Spark 기반 분석 플랫폼
      • Apache Spark란?
        1. In memory 기반의 고속 데이터 처리엔진
        2. 기존 Hadoop이 디스크 기반으로 수행하는 속도적인 면을 개선
        3. 범용저긴 분산 환경의 고성능 클러스터링 플랫폼
    2. Azure Bot Services
      • 서버를 사용하지 않는 지능형 봇 서비스로, 필요할 때 확장 가능
      • Cognitive Services를 사용하여 봇에 인텔리전스 제공
      • 상거래 챗봇, 정보 챗봇, 엔터프라이즈 생산성 챗봇 등에 활용
      • Microsoft Bot Framework 커넥터 및 BotBuilder SDK 를 제공하여 개발에 편의를 제공.
      • 웹 사이트, 앱, Cortana, Microsoft Teams, Skype, Slack, Facebook Messenger 등에 적용
    3. Machine Learning Services
      • 실험 및 모델 관리를 갖춘 종단 간 확장 가능한 신뢰할 수 있는 플랫폼으로 모든 사용자가 AI를 접할 수 있도록 구현
      • Azure Machine Learning Packages로 데이터 과학 생산성 증대 가능.
      • Computer Vision, 텍스트 분석 및 시계열 예측 가능
      • Apache Spark와 가상 머신을 통해 확장이 가능하여 실험 속도를 증가 시킬 수 있음.
    4. Azure Machine Learning Studio
      • 브라우저 기반 Machine Learning 솔루션.
      • 시각적인 환경으로 코딩이 필요하지 않음.
      • 예측 분석 솔루션을 쉽게 빌드, 배포 및 공유할 수 있도록 하는 완전 관리형 클라우드 서비스로
      • 사용자 지정 코드에 대한 수백 가지의 기본 제공 패키지 및 지원이 포함.
    5. Azure Search
      • 웹앱 및 모바일 앱 개발을 위한 AI 지원 클라우드 검색 서비스
      • 인지 기술을 통해 인사이트 보강 및 추출
      • 쉽게 확장 및 축소
      • 더욱 효과적인 검색 순위 조정으로 검색 결과와 비즈니스 목표를 연결
      • Microsoft의 심층적인 자연어 처리 노하우 활용
      • 앱에 지리 공간 검색을 간단히 추가
      • 자연어 처리의 경우 아직 인지-NLP가 한국어를 완벽하게 지원하지 않는 관계로 한국어 처리가 부족할 수 있음.
    6. Cognitive Service

    시각

    1. Computer Vision API
      • 이미지에서 다양한 정보를 추출하여 시각적 데이터를 분류 및 처리하고 이미지의 기계 지원 수정을 수행하여 서비스 조정을 지원
      • 이미지 분류, 이미지의 장면 및 활동 인식, 이미지의 유명 인사 및 이정표 인식, 이미지의 OCR(광학 문자 인식), 필기 인식 등의 기능 제공
      • 한국에서는 API가 제공되지 않음.
    2. Video Indexer
      • 비디오의 얼굴 감지
      • 비디오의 개체, 장면 및 활동 감지
      • 메타데이터, 오디오, 키 프레임 추출 및 분석
    3. Custom Vision
      • 사용자 지정 가능한 이미지 인식
      • 이미지 업로드 후 학습 시켜 REST API를 통해 분석 가능
    4. Face
      • 이미지의 얼굴 감지
      • 이미지의 사람 식별
      • 이미지의 감정 인식
      • 이미지의 유사한 얼굴 식별 및 그룹화
      • 한국에서는 API가 제공되지 않음.
    5. Content Moderator
      • 이미지, 텍스트 및 동영상에 대한 컴퓨터 지원 콘텐츠 조정 API 및 사용자 검토 도구
      • 잠재적인 불쾌감을 주는 이미지 및 원치 않는 이미지 검색
      • 잠재적인 불경한 언어 및 바람직하지 않은 텍스트 필터링
      • 동영상의 성인/외설 콘텐츠 조정

    음성

    1. 음성 텍스트 변환
      • 자동 음성 인식 및 음성텍스트 변환
      • 사용자 지정 가능한 음성 인식 및 음성 텍스트 변환
      • 고유한 어휘 또는 액센트를 위한 사용자 지정 가능한 음성 모델
      • 영어, 중국어, 불어, 독일어, 이탈리아어, 스페인어 지원
    2. Text to Speech
      • 자동 음성 텍스트 변환
      • 음성 텍스트 변환의 사용자 지정 가능한 음성 글꼴
      • Cognitive Services 음성 서비스의 일부인 Text to Speech API는 거의 실시간으로 텍스트를 오디오로 변환하여 고객의 접근성과 유용성을 향상
      • 남성/여성 음성 옵션을 포함하여 45개 이상 언어 또는 로캘의 75개 이상 음성 중에서 선택하고 속도, 피치, 볼륨, 발음, 추가 일시 중지 등의 매개 변수를 조정 가능
    3. 화자 인식
      • 화자 식별
      • 화자 검증
      • 음성을 통해 인증 수단 활용 가능
    4. 음성 번역
      • 개발자가 종단 간 실시간 음성 번역을 응용 프로그램이나 서비스에 추가할 수 있는 클라우드 기반 자동 번역 서비스

    언어

    1. 텍스트 분석
      • 텍스트에서 감정, 핵심 구 및 언어 검색
      • 감상 분석 - 한국어 지원 X
      • 핵심 문구 추출 - 입력 텍스트의 핵심 요지를 나타내는 문자열 목록을 반환, 한국어 지원 O
      • 언어 감지- 검색된 언어 및 0~1 사이의 숫자 점수를 반환. 1에 가까울 수록 확실함. 총 120개 언어 지원.
      • 텍스트에서 엔터티 식별 - 조직, 사람, 위치 등 텍스트에서 모든 명명된 엔터티를 검색. 엔터티 링크 설정은 텍스트를 웹의 추가 정보에 연결하여 개별 엔터티를 명확하게 구분. 영어만 지원.
    2. Bing Spell Check
      • 웹 규모의 다국어 맞춤법 검사
      • 상황에 맞는 맞춤법 검사
    3. Language Understanding
      • 앱, 봇 및 IoT 장치에 자연어 인식 기능을 빌드하기 위한 기계 학습 기반 서비스.
      • 자연어 처리 서비스
      • 한국어 지원 하지 않음.
    4. Translator Text
      • 자동 언어 감지
      • 자동화된 텍스트 번역
      • 사용자 지정 가능한 번역
      • REST API 지원으로 사용성이 높음.
      • 60개 언어 지원

    지식

    1. QnA Maker
      • 구조화되지 않은 텍스트에서 Q&A 추출
      • Q&A 컬렉션을 통한 기술 자료 만들기
      • 기술 자료에 대한 의미 체계 일치
      • 영어, 불어, 이탈리아어, 독일어, 스페인어 지원
    2. Custom Decision
      • 클라우드 기반의 상황에 따른 의사 결정 API
      • 제공된 정보의 컨텍스트를 이해하고 옵션의 순위를 매겨 결정
      • 피드백에 따라 자동으로 최적화

    Search

    1. Bing Web Search
      • 광고 없는 웹 검색, 안전한 웹 검색, 위치 인식 웹 검색
    2. Bing Custom Search
      • 사용자 지정 검색 엔진 만들기
      • 광고 없는 사용자 지정 검색 결과
      • 토픽 사이트 및 이미지 식별, 자동 쿼리 제안 제공, Bing 우선 순위 적용, 관련 검색 결과 제공
    3. Bing video Search
      • 광고 없는 비디오 검색
      • 비디오 토픽 및 추세 식별
      • 작성자, 인코딩 형식, 비디오 길이, 조회 수, 향상되고 간소화된 페이징 등의 유용한 메타데이터를 제공
    4. Bing Image Search
      • 광고 없는 이미지 검색
      • 미리 보기, 전체 이미지 URL, 게시 웹 사이트 정보, 이미지 메타데이터 등의 결과 제공
    5. Bing Visual Search
      • 이미지 식별 및 분류
      • 이미지를 통한 정보 취득
      • 유사한 이미지 식별
      • 이미지를 해석하여 적합한 시각적 환경 빌드
      • 유명 인사, 기념물, 아트워크 및 기타 관련 개체를 인식
      • 단일 API를 통해 이미지의 바코드를 식별하거나 텍스트 정보를 추출
    6. Bing Entity Search
      • 가장 관련 있는 엔터티 검색
      • 검색된 용어에 따라 가장 관련 있는 엔터티를 식별하고 유명한 사람, 장소, 영화, TV 프로그램, 비디오 게임, 도서, 가까운 현지 업무 등 여러 엔터티 형식을 포함
    7. Bing News Search
      • 광고 없는 뉴스 검색 결과
      • 추세 토픽 식별
      • 뉴스 기사의 권한 있는 이미지, 관련 뉴스 및 범주, 공급자 정보, 기사 URL, 추가된 날짜 등과 같은 세부 정보 제공
    8. Bing Autosuggest
      • 검색 쿼리 자동 완성
      • 앱 또는 웹 사이트에 지능형 자동 완성 기능을 추가
